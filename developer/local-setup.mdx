## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js 18+** — Download from [nodejs.org](https://nodejs.org)
- **MongoDB** — A running MongoDB instance (MongoDB Atlas free tier recommended)
- **Git** — For cloning the repository

## Quick Start

<Steps>

### Clone the repository

```bash
git clone https://github.com/dannymang/sjsuricepurity.git
cd sjsuricepurity
```

### Install dependencies

<CodeGroup>

```bash npm
npm install
```

```bash yarn
yarn install
```

```bash pnpm
pnpm install
```

</CodeGroup>

### Set up environment variables

Create a `.env.local` file in the project root with your MongoDB connection string. See the [Environment Variables](/developer/environment-variables) page for detailed instructions.

### Run the development server

<CodeGroup>

```bash npm
npm run dev
```

```bash yarn
yarn dev
```

```bash pnpm
pnpm dev
```

</CodeGroup>

### Open the application

Navigate to [http://localhost:3000](http://localhost:3000) in your browser to see the application running.

</Steps>

## Available Scripts

The project includes the following npm scripts:

```json
{
  "dev": "next dev --turbopack",
  "build": "next build",
  "start": "next start",
  "lint": "next lint"
}
```

| Script | Description |
|--------|-------------|
| `dev` | Starts the development server with Turbopack |
| `build` | Builds the application for production |
| `start` | Starts the production server |
| `lint` | Runs ESLint to check for code issues |

<Note>

The development server uses **Turbopack** for faster hot module replacement. This provides near-instant updates when you save changes to your code.

</Note>
