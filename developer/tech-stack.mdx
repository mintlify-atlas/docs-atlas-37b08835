# Tech Stack

The SJSU Purity Test is built with a modern, minimal tech stack optimized for fast development and simple deployment.

<CardGroup cols={2}>
  <Card title="Next.js 15" href="https://nextjs.org">
    App Router with file-based routing, Turbopack for fast builds, and React Server Components for API routes. Provides the foundation for both UI and backend in a single framework.
  </Card>

  <Card title="React 19" href="https://react.dev">
    Latest React with `useState` for answer state management, `useRouter` for client-side navigation, and `Suspense` boundaries for async components like the results page.
  </Card>

  <Card title="TypeScript" href="https://www.typescriptlang.org">
    Strict typing throughout the codebase. Interfaces for stats data, proper typing for Mongoose connections, and full type safety from frontend to backend.
  </Card>

  <Card title="Mongoose 8" href="https://mongoosejs.com">
    ODM for MongoDB with schema definition and model creation. Includes connection caching pattern to prevent multiple connections during development hot reloads.
  </Card>

  <Card title="MongoDB" href="https://www.mongodb.com">
    Document store for test results. Each submission stores the score, answers array, and timestamp. Aggregation pipelines used for statistics calculation.
  </Card>

  <Card title="Tailwind CSS 4" href="https://tailwindcss.com">
    Utility-first CSS framework for styling. Configured via `@tailwindcss/postcss` for the new v4 architecture.
  </Card>
</CardGroup>
